## Структурный аудит: currentPrompt + отсчёт на табло (T15-риск)

**Риск, обнаруженный ревьюером T15:** в `board/App.svelte` блоки `currentPrompt` и
счётчика ANSWERING были независимы — оба могли показываться одновременно.

**Факт после патча:** в коммите этого таска исправлен `web/src/board/App.svelte` (строка 31):

```svelte
<!-- ДО (дефект): -->
{#if state.currentPrompt}                        <!-- показывается всегда -->

<!-- ПОСЛЕ (исправлено): -->
{#if state.currentPrompt && !(state.phase === 'ANSWERING' && state.answeringTeamId)}
```

Скриншот `test-results/answering/board-answering.png` подтверждает: в фазе ANSWERING
виден только блок отсчёта («ОТВЕЧАЕТ КОМАНДА А»), `currentPrompt` скрыт.
E2E-тест `табло: при ANSWERING prompt скрыт, countdown виден` — PASS.

**Итог: структурный дефект устранён.**
